Pràctiques de TDIW

Accés a la pàgina web:

https://tdiw-g5.deic-docencia.uab.cat

Qualsevol usuari amb un password correcte pot accedir a la pàgina web, tret que ho configuris d'una altra forma fent servir un arxiu .htaccess


Accés a la base de dades:

Accés a la base de dades fent servir un plugin del Visual Studio Code: https://marketplace.visualstudio.com/items?itemName=cweijan.vscode-database-client2

Cal que instal.leu aquesta extensió quan hagueu establert una connexió remota del Visual Studio cap a les màquines del laboratori. D'aquesta manera el tràfic que generarà el client de base de dades serà des de dins de la xarxa del laboratori cap al servidor.

Les dades necessàries per realitzar la connexió son:

Server Type: PostgreSQL

Host: deic-docencia.uab.cat

port: 5432

UserName: tdiw-g5

Accés als logs:

https://tdiw-g5.deic-docencia.uab.cat/logs

Trobareu els darrers logs en format text pla. Només l'usuari "amo dels logs" els pots consultar.
